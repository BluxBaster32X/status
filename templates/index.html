Uber Eats Discord Bot

Bot Status

The Discord bot is running. You can control it using the commands in your Discord server.

Available Commands

Command Description
!open Set ordering status to OPEN
!close Set ordering status to CLOSED
!status Display the current ordering status
!help Show help information

Configuration

The bot is configured using environment variables in the .env file:

  • Discord Bot Token Required
  • Webhook URL Optional
  • Announcement Channel ID Optional
  • Authorized Users Required

For more information, check the GitHub repository.
